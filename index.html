Hello, I'm Riddhibrata.

Valley 02 — The Trail

How I find the way through.

01 · Discover

Listen before drawing

User interviews, support-ticket mining, analytics deep-dives. I map where travellers get lost before proposing new roads.

02 · Define

Chart the terrain

Journey maps, jobs-to-be-done, success metrics agreed with product and engineering — so we all climb the same mountain.

03 · Design

Build the bridge

Flows → wireframes → high-fidelity prototypes in Figma. Motion and micro-interactions are part of the design, not decoration.

Usability testing, dev handoff with tokens and specs, and post-launch measurement. Shipped is the start, not the finish line.
